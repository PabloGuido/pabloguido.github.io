Laburando el Currículum

The goal of this project was to gamify the experience of creating a resume. The gameplay consisted of showing players different resumes containing mistakes, which they had to identify and correct.
The game was developed specifically for "Expo Joven", the largest employability event in Latin America, which took place on July 5th and 6th, 2023.
Emmyta was in charge of the graphic design.

Engine: Unity (C#)

Capacihartate

Capacihartate is a puzzle game where you control two characters at the same time. Complete the tasks to access the next level and reveal the story.
This game was developed for the BAJoven GameJam, under the theme of "Economic Empowerment", in the context of Women's Month. And it intends to put into perspective an example of an experience that femininities -or dissidents- could experience in a work environment where different gender inequities are present.
This game was developed together with Nicolás Barreras

Engine: Defold (Lua)

Typing window

This is a game where the objective is to write the words that appear on screen before the time runs out. There are two types of words, some that can be written in any order and others where the order indicated by the blanks must be followed.
It can be played in Spanish or English.

This is a version I made of the well know game "2048".
